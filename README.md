# Coding Challenge 07 — Mobile Co. Landing Page

A responsive landing page for a fictional cell phone company, built with plain
HTML, CSS, and JavaScript.
## Features

- Semantic HTML layout: `header`, `nav`, `main`, `section`, `footer`
- All-black theme with white accents — no boxes, clean minimal design
- **Flexbox** header and **CSS Grid** plan layout
- Hover effects on nav links, plan titles (growing underline), and the button
- Animated CSS signal bars in the header
- Fully responsive via `@media` queries (desktop, tablet, mobile)
- **Live DOM interaction:** type a new headline and the hero `<h2>` updates instantly

## Files

| File | Purpose |
|------|---------|
| `index.html` | Page structure and content |
| `style.css`  | Layout, color scheme, animations, responsiveness |
| `cc_7.js`    | DOM manipulation for the customizable headline |

## How to run

Open `index.html` in any web browser — no build step or server required.
